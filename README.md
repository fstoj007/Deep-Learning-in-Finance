# Deep-Learning-in-Finance
This repository contains a structured collection of lecture materials for the course
**“Mathematics for New Technologies in Finance”**, taught by Josef Teichmann at ETH Zurich.

---

## Overview

The course develops mathematical foundations and modern methodologies at the interface of the following fields,

* Machine learning,
* Stochastic analysis,
* Mathematical finance,

with a particular emphasis on **deep learning methods in financial applications**.

This repository aggregates the lecture notebooks into a single, coherent GitHub structure for ease of access and study.

---

## Source Material

All materials originate from the official course webpage:

* https://people.math.ethz.ch/~jteichma/index.php?content=teach_mlf2026

The original content is maintained by the course instructor. This repository is **not an official distribution**, but rather a curated mirror.

## Usage

To work with the materials locally:

```bash
git clone https://github.com/fstoj007/Deep-Learning-in-Finance.git
cd ethz-mathematics-deep-learning-finance
```

Install dependencies (if provided):

```bash
pip install -r requirements.txt
```

Then launch Jupyter:

```bash
jupyter notebook
```
